TOEIC Test Chousoku Training

TOEIC Test Chousoku Training is an educational video game developed by Mechanic Arms for the Nintendo 3DS. It was released on April 5, 2012 only in Japan and is designed as an interactive application to improve English language proficiency based on the standardized TOEIC system.

Mii usage

Mii characters are required for this title, serving as save file icons and avatars rendered in 3D within its main menu. Male Miis will wear a blue shirt, and female Miis will wear a red shirt.

Two custom guest Miis are used instead of the six included with the Face Library.
